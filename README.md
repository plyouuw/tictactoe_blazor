# Gra Kółko i Krzyżyk napisana w języku C# we frameworku Blazor. 
Zawiera tryb sieciowego multiplayer, możlwiość grania na planszy NxM, oraz bota z którym możemy się zmierzyć. 
Gra posiada możliwość tworzenia wielu pokoi, do których mogą dołączać gracze, jak i osoby zainteresowane (obserwatorzy). 

Wykorzystuje technologię SignalR z HubConnetions
